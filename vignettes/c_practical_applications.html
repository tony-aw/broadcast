Introduction

Broadcasting comes up frequent enough in real world problems. This articles gives a few examples of these.

 

Vector quantization

This first example is taken from Numpy’s own documentation.

The basic operation in Vector Quantization (VQ) finds the closest point in a set of points, called codes in VQ jargon, to a given point, called the observation. In the very simple, two-dimensional case shown below, the values in observation describe the weight and height of an athlete to be classified. The codes represent different classes of athletes. Finding the closest point requires calculating the distance between observation and each of the codes. The shortest distance provides the best match. In this example, codes[0] is the closest class indicating that the athlete is likely a basketball player.


observation <- array(c(111.0, 188.0), dim = c(1, 2))
codes <- array(
  c(102.0, 203.0,
    132.0, 193.0,
    45.0, 155.0,
    57.0, 173.0),
  dim = c(4, 2)
)

diff <- bc.num(codes, observation, "-")
dist <- apply(diff^2, 2L, sum) |> sqrt()
which.min(dist); dist[which.min(dist)]
#> [1] 1
#> [1] 125.3395

 

Find most resembling string

If you have 2 vectors of strings, and you want to find which pair of strings (one string from each vector), you’ll need to find the string distance of all possible combinations.

This can be done very fast and very conveniently with ‘broadcast’, using the following code:

x <- array(month.name)
y <- array(month.abb, c(1, 12))

out <- bc.str(x, y, "levenshtein")
dimnames(out) <- list(x, y)
print(out)
